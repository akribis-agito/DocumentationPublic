# Syntax

For this document, ASCII encoding will be used for assignment and query of the keywords. For more information on the ASCII encoding and syntax, please refer to Agito Communication Manual.

Throughout this document, the syntax for each keyword is described with placeholders. The list of placeholders is as shown.

| No. | Type of placeholder     | Placeholder | Syntax example                | Message example       |
| --- | ----------------------- | ----------- | ----------------------------- | --------------------- |
| 1   | Axis placeholder        | ?           | ?CalcFilters                  | ACalcFilters          |
| 2   | Argument placeholder    | Value       | ?EncDir = Value               | DEncDir = 1           |
| 3   | Array index placeholder | Index       | ?SinCosSetup\[Index\] = Value | ESinCosSetup\[3\] = 4 |
