# IndirectArray

**Definition:**

IndirectArray specifies the CAN code of the target array parameter to be accessed indirectly. Together with IndirectIndex and IndirectValue, it forms the three-register indirect access mechanism that allows dynamic parameter array addressing. It is a non-axis parameter and is not saved to flash.
