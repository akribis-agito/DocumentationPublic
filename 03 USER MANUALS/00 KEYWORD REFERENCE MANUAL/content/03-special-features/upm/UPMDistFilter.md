# UPMDistFilter

**Definition:**

UPMDistFilter sets the low-pass filter cutoff applied to the disturbance estimate within the UPM rejection loop, smoothing the adaptive correction to prevent noise amplification. It cannot be changed while the axis is in motion or with the motor on. It is an axis-related parameter saved to flash.
