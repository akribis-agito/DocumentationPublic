# UPMRptMotion

**Definition:**

UPMRptMotion selects the motion trigger mode that signals the start of each repetitive motion cycle to the UPM compensation algorithm, enabling it to synchronize the correction table with the trajectory. It cannot be changed while the axis is in motion; it can be changed with the motor on. It is an axis-related parameter and is not saved to flash.
