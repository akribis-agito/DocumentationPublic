# UPMRptOn

**Definition:**

UPMRptOn enables the UPM repetitive (periodic) motion compensation feature. When active, the controller adapts a feedforward correction to cancel position errors that repeat with each motion cycle. It cannot be changed while the axis is in motion; it can be changed with the motor on. It is an axis-related parameter and is not saved to flash.
