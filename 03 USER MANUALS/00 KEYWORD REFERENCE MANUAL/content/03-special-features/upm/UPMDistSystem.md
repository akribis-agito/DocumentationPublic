# UPMDistSystem

**Definition:**

UPMDistSystem selects the plant model or disturbance frequency reference used by the UPM disturbance rejection algorithm to identify the harmonic components to be suppressed. It cannot be changed while the axis is in motion or with the motor on. It is an axis-related parameter saved to flash.
