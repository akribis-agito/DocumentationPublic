# UPMDistOn

**Definition:**

UPMDistOn enables the UPM (Universal Performance Monitor) disturbance rejection function. When active, the controller applies an adaptive filter to cancel periodic disturbances at the frequencies identified by UPMDistSystem. It is an axis-related parameter and is not saved to flash; it can be changed at any time.
