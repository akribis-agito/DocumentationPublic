# UPMVelOn

**Definition:**

UPMVelOn enables the UPM velocity-based compensation function, which applies a velocity-dependent feedforward correction table (UPMVelTable) to reduce friction and viscous drag errors across the operating velocity range. It is an axis-related parameter and is not saved to flash; it can be changed at any time.
