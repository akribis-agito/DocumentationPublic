# UPMRptTime

**Definition:**

UPMRptTime sets the period of the repetitive motion cycle in milliseconds used by the UPM repetitive compensation algorithm to align the correction table with the motion pattern. It cannot be changed while the axis is in motion; it can be changed with the motor on. It is an axis-related parameter saved to flash.
