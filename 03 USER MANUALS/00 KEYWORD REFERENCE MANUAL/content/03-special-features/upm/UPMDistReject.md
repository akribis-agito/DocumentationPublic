# UPMDistReject

**Definition:**

UPMDistReject sets the rejection bandwidth or gain of the UPM disturbance cancellation filter, controlling how aggressively periodic disturbances are attenuated relative to the residual noise floor. It is an axis-related parameter saved to flash and can be changed at any time.
