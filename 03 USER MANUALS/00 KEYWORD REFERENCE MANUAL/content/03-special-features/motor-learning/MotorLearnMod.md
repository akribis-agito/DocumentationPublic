# MotorLearnMod

**Definition:**

MotorLearnMod selects the learning mode, determining which type of motor non-linearity (such as torque ripple or cogging) the learning algorithm targets and how the resulting correction table is structured. It cannot be changed while the axis is in motion; it can be changed with the motor on. It is an axis-related parameter and is not saved to flash.
