# MotorLearnOn

**Definition:**

MotorLearnOn enables the motor-learning routine, which drives the motor open-loop (advancing the commutation electrical angle in MotorLearnInc steps, per the mode selected by MotorLearnMod) to measure and report the motor pole pairs (MotorLearnPl) and the encoder resolution (MotorLearnRes); progress is reported by MotorLearnSta. It cannot be changed while the axis is in motion; it can be changed with the motor on. It is an axis-related parameter and is not saved to flash.
