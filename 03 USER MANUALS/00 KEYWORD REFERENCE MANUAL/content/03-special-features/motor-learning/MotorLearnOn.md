# MotorLearnOn

**Definition:**

MotorLearnOn enables the motor learning process that measures and records the motor's torque ripple or commutation non-linearity for subsequent compensation. It cannot be changed while the axis is in motion; it can be changed with the motor on. It is an axis-related parameter and is not saved to flash.
