# MotorLearnInc

**Definition:**

MotorLearnInc sets the incremental update step size for the motor learning algorithm, controlling how aggressively each measured sample updates the correction table. It cannot be changed while the axis is in motion or with the motor on. It is an axis-related parameter saved to flash.
