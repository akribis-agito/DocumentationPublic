# AutoGNumSet

**Definition:**

AutoGNumSet sets the number of valid identification segments that must be collected and averaged before the auto-gain algorithm finalizes its inertia estimate and computes updated gains. It is an axis-related parameter saved to flash and can be changed at any time.
