# AutoGDownSm

**Definition:**

AutoGDownSm sets the downsampling factor applied to the motion data collected during auto-gain identification, reducing computation load at the cost of frequency resolution. It cannot be changed while the axis is in motion or with the motor on. It is an axis-related parameter saved to flash.
