# AutoGMinLen

**Definition:**

AutoGMinLen sets the minimum length in servo cycles of a motion segment that the auto-gain algorithm will accept for inertia identification. Segments shorter than this threshold are discarded. It is an axis-related parameter saved to flash and can be changed at any time.
