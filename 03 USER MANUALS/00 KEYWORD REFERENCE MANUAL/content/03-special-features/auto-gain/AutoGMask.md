# AutoGMask

**Definition:**

AutoGMask is an array that specifies which servo gain parameters the auto-gain algorithm is allowed to update when AutoGCopy is executed. Setting an element to zero prevents the corresponding gain from being overwritten. It is an axis-related array parameter saved to flash and can be changed at any time.
