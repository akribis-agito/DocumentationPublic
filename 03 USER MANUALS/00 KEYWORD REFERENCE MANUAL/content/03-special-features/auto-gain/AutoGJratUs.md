# AutoGJratUs

**Definition:**

AutoGJratUs sets the user-supplied inertia ratio used by the automatic gain tuning algorithm to constrain the gain computation when the system inertia is not fully identified. It is an axis-related parameter saved to flash and can be changed at any time.
