# AutoGMaxRat

**Definition:**

AutoGMaxRat sets the maximum acceptable inertia ratio threshold used by the auto-gain algorithm to validate identification results. Estimates above this ratio are rejected as anomalous. It is an axis-related parameter saved to flash and can be changed at any time.
