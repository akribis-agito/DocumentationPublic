# AutoGMaxLen

**Definition:**

AutoGMaxLen sets the maximum length in servo cycles of a motion segment that the auto-gain algorithm will process. Segments exceeding this length are truncated to limit computation time. It is an axis-related parameter saved to flash and can be changed at any time.
