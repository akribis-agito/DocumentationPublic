# AutoGMinRat

**Definition:**

AutoGMinRat sets the minimum acceptable inertia ratio threshold used by the auto-gain algorithm to validate identification results. Estimates below this ratio are rejected as unreliable. It is an axis-related parameter saved to flash and can be changed at any time.
