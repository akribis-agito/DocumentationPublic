# AntiCoggValue

*Legacy keywords*

**Definition:**

AntiCoggValue was the array of precomputed cogging torque correction values used by the anti-cogging compensation feature. This keyword was removed in a later firmware release and is no longer available; the related functionality was superseded by anti-disturbance functionality.
