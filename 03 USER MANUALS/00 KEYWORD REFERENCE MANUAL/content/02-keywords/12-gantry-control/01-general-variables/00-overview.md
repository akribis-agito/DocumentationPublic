# General variables

General gantry-control variables: enabling gantry MIMO mode and the yaw/error-map correction parameters.

- [GantryOn](GantryOn.md) — enables gantry MIMO control on the A axis
- [GantryDLoopOn](GantryDLoopOn.md) — enables dual-loop gantry control
- [GantryYawRef](GantryYawRef.md) — yaw correction reference
- [GantryCurrRef](GantryCurrRef.md) — yaw current (torque) reference
- [GantryMap](GantryMap.md) / [GantryMapInit](GantryMapInit.md) / [GantryMapSrc](GantryMapSrc.md) / [GantryMapType](GantryMapType.md) / [GantryMapVal](GantryMapVal.md) — error-map correction
