# General variables

General gantry-control variables: enabling gantry MIMO mode and the position-dependent decoupling-map parameters.

- [GantryOn](GantryOn.md) — enables gantry MIMO control on the A axis
- [GantryDLoopOn](GantryDLoopOn.md) — enables dual-loop gantry control (central-i v5)
- [GantryYawRef](GantryYawRef.md) — yaw correction reference
- [GantryCurrRef](GantryCurrRef.md) — yaw current (torque) reference
- [GantryMap](GantryMap.md) / [GantryMapGap](GantryMapGap.md) / [GantryMapInit](GantryMapInit.md) / [GantryMapSrc](GantryMapSrc.md) / [GantryMapType](GantryMapType.md) / [GantryMapVal](GantryMapVal.md) — position-dependent decoupling map (central-i v5)
- [GantrySwapSrc](GantrySwapSrc.md) — dual-loop swap source pointer (central-i v5)
