# GantryAuxFdbk

**Definition:**

GantryAuxFdbk is a read-only parameter that reports the feedback position from the auxiliary encoder of the gantry axis. It is used by the gantry dual-loop controller to measure the yaw (misalignment) angle between the two beam ends. It is an axis-related status variable.
