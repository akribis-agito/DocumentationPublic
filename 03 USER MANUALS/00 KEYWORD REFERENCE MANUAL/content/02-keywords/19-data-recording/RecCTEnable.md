# RecCTEnable

**Definition:**

RecCTEnable is an array parameter that enables or disables the continuous-time recording mode for each recording channel. When enabled, the channel captures data at every servo cycle up to the size set by RecCTMaxSize. It is a non-axis parameter saved to flash.
