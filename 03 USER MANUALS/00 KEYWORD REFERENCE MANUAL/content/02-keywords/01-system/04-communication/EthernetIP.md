# EthernetIP

**Definition:**

In Ethernet communication, EthernetIP is a parameter array storing the IP address.

For example, if IP addresss is “172.1.1.50”, each array element is as follows.

| Parameter        | Value |
| ---------------- | ----- |
| EthernetIP\[1\]  | 172   |
| EthernetIP\[2\]  | 1     |
| EthernetIP\[3\]  | 1     |
| EthernetIP\[4\]  | 50    |
