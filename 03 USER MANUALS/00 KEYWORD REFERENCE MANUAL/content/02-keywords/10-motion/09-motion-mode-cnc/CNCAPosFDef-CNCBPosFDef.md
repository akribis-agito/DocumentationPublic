# CNCAPosFDef/CNCBPosFDef

**Definition:**

CNCAPosFDef (and its CNCB equivalent) is an array that defines the position filter configuration for the CNC motion queue A (or B). Each element specifies filter parameters applied to the position reference of CNC segments. It is a non-axis array saved to flash and can be changed at any time.
