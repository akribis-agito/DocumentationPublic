# CNCAEndSegMod/CNCBEndSegMod

**Definition:**

CNCAEndSegMod (and its CNCB equivalent) selects the end-of-segment behaviour for CNC motion queue A (or B), controlling how the controller handles the transition between consecutive CNC segments (for example, blending vs. stopping). It is a non-axis parameter saved to flash and can be changed at any time.
