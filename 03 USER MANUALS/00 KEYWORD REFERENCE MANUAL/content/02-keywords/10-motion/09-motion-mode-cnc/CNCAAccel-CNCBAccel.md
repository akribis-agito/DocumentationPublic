# CNCAAccel/CNCBAccel

**Definition:**

CNCAAccel (and its CNCB equivalent) is a read-only parameter that reports the current acceleration value of the active CNC motion segment being executed on queue A (or B). It reflects the acceleration encoded in the segment pushed to the CNC FIFO. It is a non-axis, read-only parameter that is not saved to flash.
