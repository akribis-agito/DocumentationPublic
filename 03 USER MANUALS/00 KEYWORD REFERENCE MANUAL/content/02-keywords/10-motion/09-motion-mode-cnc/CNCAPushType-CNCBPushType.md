# CNCAPushType/CNCBPushType

**Definition:**

CNCAPushType (and its CNCB equivalent) is a command used to push a new motion segment into the CNC FIFO queue A (or B). The argument selects the segment type (for example, linear or arc) and the segment parameters are taken from the associated CNC parameter set. It is a non-axis command function that can be issued at any time.
