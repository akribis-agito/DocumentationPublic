# CNCAEndSpeed/CNCBEndSpeed

**Definition:**

CNCAEndSpeed (and its CNCB equivalent) is a read-only parameter that reports the commanded speed at the end of the currently executing CNC motion segment on queue A (or B). It is a non-axis, read-only parameter that is not saved to flash.
