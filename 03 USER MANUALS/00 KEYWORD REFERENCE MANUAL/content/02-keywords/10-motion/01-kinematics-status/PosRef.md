# PosRef

Position reference into the position loop, after profiler post-processing.

## Overview

`PosRef` is the position reference, in main user units, after the optional post-processing steps (offset, moving average, input shaping, injection and filter) applied to the value generated by the motion profiler. It is the input to the position loop. The position error [PosErr](PosErr.md) is derived from `PosRef`, and the velocity reference [dPosRef](dPosRef.md) is its filtered derivative.

## Examples

```text
PosRef?             ; read the current position reference
```
