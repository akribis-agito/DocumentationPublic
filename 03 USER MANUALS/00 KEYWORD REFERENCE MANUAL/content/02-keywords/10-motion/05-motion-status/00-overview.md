# Motion status

Motion status is updated while the motion is underway. Some keyword’s status behaviour can also be configured by other keywords described below. The table below shows the summary of motion status keywords.

| No. | Keywords      | Summary |
|-----|---------------|---------|
| 1   | InTargetStat  |         |
| 2   | InTargetTime  |         |
| 3   | InTargetTol   |         |
| 4   | InTargetVelTh |         |
| 5   | MotionReason  |         |
| 6   | MotionSamples |         |
| 7   | MotionStat    |         |
| 8   | RptCounter    |         |
