# VecEncFactDn

**Definition:**

VecEncFactDn is the denominator of the encoder scaling ratio applied to the vector motion position reference for this axis. Together with VecEncFactNu it defines the rational scale (VecEncFactNu / VecEncFactDn) mapping vector position units to axis encoder counts. It is an axis-related parameter.
