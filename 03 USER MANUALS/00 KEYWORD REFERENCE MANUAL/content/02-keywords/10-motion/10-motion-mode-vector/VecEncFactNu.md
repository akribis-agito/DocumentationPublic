# VecEncFactNu

**Definition:**

VecEncFactNu is the numerator of the encoder scaling ratio applied to the vector motion position reference for this axis. The effective scale is VecEncFactNu / VecEncFactDn, mapping the vector position units to the axis encoder counts. It is an axis-related parameter.
