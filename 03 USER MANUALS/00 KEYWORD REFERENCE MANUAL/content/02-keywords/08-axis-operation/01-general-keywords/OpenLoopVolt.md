# OpenLoopVolt

Voltage reference applied to the modulation in voltage open-loop mode.

## Overview

`OpenLoopVolt` is the voltage reference value applied onto the space vector modulation (for PWM drive) while the axis is in the voltage open-loop condition. It is only used when [OpenLoopOn](OpenLoopOn.md) = 2, and it bypasses all voltage references generated by the current control loop.

> **Note:** Please contact Agito for more information if this feature is needed in your application.

## Examples

```text
AOpenLoopOn=2        ; enter voltage open loop
AOpenLoopVolt=500    ; apply voltage reference
```
