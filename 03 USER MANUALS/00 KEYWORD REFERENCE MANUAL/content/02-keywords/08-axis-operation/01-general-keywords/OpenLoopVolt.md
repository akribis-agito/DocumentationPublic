# OpenLoopVolt

**Condition:**

It is only used when OpenLoopOn = 2.

**Definition:**

OpenLoopVolt is the voltage reference value applied onto the space vector modulation (for PWM drive) if the axis is in voltage open-loop condition.

This value will bypass all voltage references generated by the current control loop.

Please contact Agito for more information if this feature is needed in application.
