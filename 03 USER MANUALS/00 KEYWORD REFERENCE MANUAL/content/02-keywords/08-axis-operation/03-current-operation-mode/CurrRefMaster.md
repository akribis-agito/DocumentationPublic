# CurrRefMaster

**Condition:**

This keyword is only applicable when CurrCmdSrc = 3.

**Definition:**

CurrRefMaster is the index of master axis (0-based indexing) where under slave drive mode, the slave axis will have its current reference (CurrRef) copied from CurrRef of master axis.

| Value | Axis |
|-------|------|
| 0     | A    |
| 1     | B    |
| …     | …    |
