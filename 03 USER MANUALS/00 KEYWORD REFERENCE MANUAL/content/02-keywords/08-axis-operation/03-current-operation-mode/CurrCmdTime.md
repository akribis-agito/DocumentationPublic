# CurrCmdTime

Command time for each current command, in milliseconds.

## Overview

`CurrCmdTime` is the command time for each current command, in milliseconds.

> **Obsolete / superseded:** This entry was imported from the 2021 PDF reference. `CurrCmdTime` is not a recognized keyword in the current firmware (neither standalone/v4 nor central-i v5). The keyword for per-entry holding time in current operation mode is [CurrCmdHTime](CurrCmdHTime.md). Use [CurrCmdHTime](CurrCmdHTime.md) instead; this page is retained only to redirect users who encounter the old name.
