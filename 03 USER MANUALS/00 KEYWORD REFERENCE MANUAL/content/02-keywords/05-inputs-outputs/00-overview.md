# Inputs/outputs

Agito supports programmable general-purpose analog and digital inputs and outputs (I/O).

The following table shows the summary of inputs/outputs keywords.

| No. | Sub-section      | Keywords                                 | Summary |
|-----|------------------|------------------------------------------|---------|
| 1   | General keywords | BiDirConfig                              |         |
| 2   | Analog inputs    | AInFilt                                  |         |
| 3   | Analog inputs    | AInOffset                                |         |
| 4   | Analog inputs    | AInDB                                    |         |
| 5   | Analog inputs    | AInGain                                  |         |
| 6   | Analog inputs    | AInMuteRange                             |         |
| 7   | Analog inputs    | AInMode                                  |         |
| 8   | Analog inputs    | AInPort                                  |         |
| 9   | Analog outputs   | AOutMode                                 |         |
| 10  | Analog outputs   | AOutOffset                               |         |
| 11  | Analog outputs   | AOutPort                                 |         |
| 12  | Analog outputs   | AOutGain                                 |         |
| 13  | Digital inputs   | DInFilt                                  |         |
| 14  | Digital inputs   | DInLog/DInLogHigh                        |         |
| 15  | Digital inputs   | DInPort/DInPortHigh                      |         |
| 16  | Digital inputs   | DInMode                                  |         |
| 17  | Digital outputs  | DOutPort                                 |         |
| 18  | Digital outputs  | DOutPortSBit/ DOutPortCBit/ DOutPortTBit |         |
| 19  | Digital outputs  | DOutMode                                 |         |
| 20  | Digital outputs  | DOutSelect                               |         |
| 21  | Digital outputs  | DOutLog                                  |         |
| 22  | Digital outputs  | DOutType                                 |         |
