# SinCosSetup/AuxSinCosSet

**Condition:**

SinCosSetup is only used when EncType=4 (SIN/COS encoder).

**Definition:**

SinCosSetup is a parameter array that defines SIN/COS encoder configuration. Each index of the array is described as follows.

**Note:**

User can enable analog test mode (TMA) using SinCosSetup[20] to check the Sin/Cos signal. Both time domain and X/Y view (Lissajous) are suitable for signal offset and phase calibration.

**Warning:**

Do not set the individual SinCosSetup value other than the values specified in the table. Any improper value may result in unexpected feedback behaviour and may cause damage to the system.

**Note:**
