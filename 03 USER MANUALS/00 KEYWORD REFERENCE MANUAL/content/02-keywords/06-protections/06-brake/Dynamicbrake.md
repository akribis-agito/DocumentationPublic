# Dynamic brake

Dynamic braking rapidly slows the motor by shorting the motor phases and bleeding off the current generated by the back-EMF. It is a safe way to stop a motor if the motor is suddenly disabled. This page covers the related keywords: `DynBrakeOn`, `DynBrkRef`, and `DynBrakeSpeed`.

## DynBrakeOn

Enables or disables dynamic braking.

| Value | Description |
|-------|-------------|
| 0 | Disabled |
| 1 | Enabled |

## DynBrkRef

Sets the maximum duty cycle used for dynamic braking. `DynBrkRef = 1000` gives the strongest braking. If the braking current exceeds the limits set by [ContCL](../02-current-and-voltage/ContCL.md) and [PeakCL](../02-current-and-voltage/PeakCL.md), the controller internally reduces the duty cycle to keep the current within limits.

## DynBrakeSpeed

Relates to the dynamic-braking behaviour described under **DynBrakeOn** above.
