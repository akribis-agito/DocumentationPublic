# MotorTempUsed

**Definition**

MotorTempUsed defines the temperature sensor type.

| Value | Descriptions                                                        |
|-------|---------------------------------------------------------------------|
| 0     | None (or if thermostat is connected to digital input)               |
| 1     | PT100                                                               |
| 2     | Thermostat (if thermostat is connected to temperature sensor input) |
